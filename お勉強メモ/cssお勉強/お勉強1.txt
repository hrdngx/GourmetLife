・cssとは
　HTMLに色や大きさや背景などを加えて見た目を良くする

・cssで記述すること
　文字の色や大きさ、フォント、背景色の変更などを記述します
　文字や画像を左寄せにするか、中央揃えにするか
　2段組みにするか
　文字と文字の間隔はどれくらいにするか、といった調節も行える

・HTML
　<h1>CSSを学習しよう！！</h1>
・CSS
  h1  <- どこの(h1要素の)
  {
    color:red; <-何を(colorを)どうするか(redにする)
  }
    　
・CSSの機能
　h1   <- セレクタ
  {
　　color:red; <- プロパティ:値
  }

・子要素のスタイルを指定
　<section>
 　<h2>サービス</h2>
 　<ul>
   　  <li>学べる！</li>
     　<li>高品質</li>
     　<li>価格</li>
 　</ul>
　</section>
　このとき、
  section ul { [プロパティ]:[値]}
  のように記述する。


・背景色の変更->background-color
　h1{
  　background-color:#f8f7f4;
　}
